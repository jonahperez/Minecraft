Installing Mods

I've struggled for many hours learning how to get mods installed on Elliot's server. Below, I've outlined the basic procedure that's worked for us. This should be the same procedure for Mac, Windows and Linux machines.

The first hurtle is making sure that you are running the appropriate version of Minecraft for the mods you want to run. We've found that the mods Elliot likes are mostly for the 1.7.10 release of Minecraft.

The good news is that the Minecraft launcher makes it easy to download previous versions of the game.

  1. Install Minecraft 1.7.10 using the following steps:
    1. Start Minecraft
    2. Click on "Edit Profile"
    3. Change the "Use version:" dropdown to 1.7.10
    4. Start the game so it downloads the 1.7.10 version
    5. Close the game
  2. Run the Forge Mod Loader installers
    1. from a command prompt run java -jar forge-1.7.10-10.13.2.1291-installer.jar
    2. Once it's complete, close the window
  3. Add the mod jar files to the mods folder under your Minecraft home folderol
    1. In Windows: c:\Users\[username]\AppData\Roaming\.minecraft\mods
    2. In Mac: ~\.minecraft\mods
    3. In Linux: ~\.minecraft\mods
